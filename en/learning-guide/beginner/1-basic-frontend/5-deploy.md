# Task 5: Deploy

[← Back to Beginner Tasks](../index.md)

Publish your resume site to the internet so anyone can see it.

## Tasks

### Option A: GitHub Pages (recommended for beginners)

- [ ] Go to your repository on GitHub
- [ ] Click **Settings → Pages**
- [ ] Under "Source", select **Deploy from a branch**
- [ ] Select **main** branch and **/ (root)** folder, then click **Save**
- [ ] Wait a few minutes, then visit `https://yourname.github.io/your-repo/`
- [ ] Confirm your resume page is live on the internet
- [ ] Share the URL with someone

### Option B: Vercel (alternative)

- [ ] Go to [vercel.com](https://vercel.com/) and sign up with your GitHub account
- [ ] Click **Add New → Project**
- [ ] Import your GitHub repository
- [ ] Click **Deploy**
- [ ] Visit the provided URL and confirm your site is live

## What You Should Understand

- Deployment means making your site accessible on the internet
- GitHub Pages hosts static sites (HTML/CSS/JS) for free
- Every time you `git push`, GitHub Pages can automatically update your site
- Your site URL is based on your GitHub username and repository name

## Congratulations!

You have completed all beginner tasks. You can now:
- Write HTML to structure a page
- Use CSS to style it
- Add interactivity with JavaScript
- Track your code with Git and GitHub
- Deploy your site for the world to see
