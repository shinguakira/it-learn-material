# Task 0: Install VS Code

[← Back to Beginner Tasks](../index.md)

Set up your development environment before writing any code.

## Tasks

- [ ] Download and install [VS Code](https://code.visualstudio.com/)
- [ ] Open VS Code and explore the interface (sidebar, terminal, explorer)
- [ ] Install the "Live Server" extension (lets you preview HTML in the browser with auto-reload)
- [ ] Create a folder for your projects (e.g., `my-projects`)
- [ ] Open the folder in VS Code using **File → Open Folder**
- [ ] Open the built-in terminal with **Ctrl + `** (or **View → Terminal**)
- [ ] Create a new file called `hello.html` and type `Hello World` in it
- [ ] Right-click the file and select **Open with Live Server** to see it in your browser

## Tips

- VS Code is free and works on Windows, Mac, and Linux
- The terminal inside VS Code is the same as your system terminal — you can run commands there
- You can install additional extensions later as needed (e.g., Prettier for formatting)
