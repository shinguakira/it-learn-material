# Task 1: HTML

[← Back to Beginner Tasks](../index.md)

> Reference: [HTML](../../../frontend/html/) — Learn about HTML structure and tags

Build your first web page using HTML.

## Tasks

- [ ] Create a new file called `index.html`
- [ ] Write the basic HTML structure (`<!DOCTYPE html>`, `<html>`, `<head>`, `<body>`)
- [ ] Add a `<title>` inside `<head>`
- [ ] Open the file in your browser (drag & drop or use Live Server)
- [ ] Create a personal resume page with:
  - [ ] `<h1>` for your name
  - [ ] `<h2>` for section titles (Contact, Summary, Education, Skills)
  - [ ] `<p>` and `<strong>` for text content
  - [ ] `<ul>` and `<li>` for a skill list
  - [ ] `<section>` to group each section
- [ ] Add `<link rel="stylesheet" href="style.css" />` inside `<head>` (we will create the CSS file in the next task)

## What You Should Understand

- `<head>` is for metadata (not visible on the page)
- `<body>` is the main visible content
- Tags come in pairs: `<tag>content</tag>` or self-closing: `<br>`, `<img />`
- `<div>` and `<section>` group content together
