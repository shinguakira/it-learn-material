# TypeScript (JavaScript)

[← Back to Frontend](../README.md)

TypeScript adds static type checking to JavaScript, helping catch errors before runtime. JavaScript concepts are covered here too — TypeScript is a superset of JavaScript.

## What You'll Learn

- Variables, data types, and operators
- Functions and scope
- DOM manipulation
- Asynchronous programming (Promises, async/await)
- ES6+ features
- Basic types and type annotations
- Interfaces and type aliases
- Generics
- Enums and utility types
- Configuring TypeScript projects
