# TypeScript（JavaScript）

[← フロントエンドに戻る](../README.md)

TypeScriptはJavaScriptに静的型チェックを追加し、実行前にエラーを検出します。JavaScriptの概念もここでカバーされます。TypeScriptはJavaScriptのスーパーセットです。

## 学習内容

- 変数、データ型、演算子
- 関数とスコープ
- DOM操作
- 非同期プログラミング（Promises、async/await）
- ES6+の機能
- 基本型と型アノテーション
- インターフェースと型エイリアス
- ジェネリクス
- 列挙型とユーティリティ型
- TypeScriptプロジェクトの設定
