# タスク0：VS Codeのインストール

[← 初級タスクに戻る](../index.md)

コードを書く前に開発環境をセットアップしましょう。

## タスク

- [ ] [VS Code](https://code.visualstudio.com/)をダウンロードしてインストールする
- [ ] VS Codeを開いて画面を確認する（サイドバー、ターミナル、エクスプローラー）
- [ ] 「Live Server」拡張機能をインストールする（ブラウザでHTMLを自動リロード付きでプレビューできる）
- [ ] プロジェクト用のフォルダを作成する（例：`my-projects`）
- [ ] **ファイル → フォルダーを開く** でフォルダをVS Codeで開く
- [ ] **Ctrl + `**（または **表示 → ターミナル**）で内蔵ターミナルを開く
- [ ] `hello.html` というファイルを作成して `Hello World` と入力する
- [ ] ファイルを右クリックして **Open with Live Server** を選択してブラウザで確認する

## ヒント

- VS Codeは無料で、Windows、Mac、Linuxで動作する
- VS Code内のターミナルはシステムのターミナルと同じ — コマンドを実行できる
- 必要に応じて後から拡張機能を追加できる（例：Prettierでフォーマット）
