# タスク1：HTML

[← 初級タスクに戻る](../index.md)

> 参照：[HTML](../../../frontend/html/) — HTMLの構造とタグを学ぶ

HTMLを使って最初のWebページを作りましょう。

## タスク

- [ ] `index.html` というファイルを作成する
- [ ] 基本的なHTML構造を書く（`<!DOCTYPE html>`、`<html>`、`<head>`、`<body>`）
- [ ] `<head>` の中に `<title>` を追加する
- [ ] ブラウザでファイルを開く（ドラッグ＆ドロップまたはLive Serverを使用）
- [ ] 個人の履歴書ページを作成する：
  - [ ] 名前に `<h1>` を使う
  - [ ] セクションタイトル（連絡先、概要、学歴、スキル）に `<h2>` を使う
  - [ ] テキストに `<p>` と `<strong>` を使う
  - [ ] スキルリストに `<ul>` と `<li>` を使う
  - [ ] 各セクションを `<section>` でグループ化する
- [ ] `<head>` 内に `<link rel="stylesheet" href="style.css" />` を追加する（CSSファイルは次のタスクで作成）

## 理解すべきポイント

- `<head>` はメタデータ用（ページには表示されない）
- `<body>` はメインの表示コンテンツ
- タグはペアで使う：`<tag>内容</tag>` または自己閉じ：`<br>`、`<img />`
- `<div>` と `<section>` はコンテンツをグループ化する
