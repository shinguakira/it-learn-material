# 中級タスク

[← 学習ガイドに戻る](../README.md)

基礎を活かして、より高度なチャレンジに取り組みましょう。

## TypeScriptタスク

> 参照：[TypeScript](../../frontend/typescript/) — 型付きJavaScriptを学ぶ

- [ ] `tsconfig.json` を使ってTypeScriptプロジェクトをセットアップする
- [ ] JavaScriptファイルを型アノテーション付きのTypeScriptに書き換える
- [ ] データモデル（例：User、Product）のインターフェースを作成する
- [ ] ユーティリティ関数でジェネリクスを使う

## Reactタスク

> 参照：[React](../../frontend/react/) — UIコンポーネントを学ぶ

- [ ] `create-react-app` またはViteでReactアプリを作成する
- [ ] 再利用可能なコンポーネント（例：Card、Button）を作る
- [ ] `useState` でコンポーネントの状態を管理する
- [ ] `useEffect` でAPIからデータを取得する
- [ ] React Routerでルーティングを追加する

## バックエンドタスク

> 参照：[Node.js](../../backend/nodejs/) | [Express](../../backend/express/) | [データベース](../../backend/databases/)

- [ ] 基本的なNode.js HTTPサーバーを作成する
- [ ] ルート付きのExpressアプリをセットアップする
- [ ] GET、POST、PUT、DELETEエンドポイントのREST APIを構築する
- [ ] データベース（PostgreSQLまたはMongoDB）に接続する
- [ ] 入力バリデーションとエラーハンドリングを追加する
