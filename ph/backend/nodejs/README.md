# Node.js

[← Bumalik sa Backend](../README.md)

Ang Node.js ay isang JavaScript runtime na binuo sa V8 engine ng Chrome para sa paggawa ng mga server-side application.

## Mga Matututunan

- Mga pangunahing kaalaman sa Node.js at event loop
- Mga module at npm
- File system at stream
- Mga pangunahing kaalaman sa HTTP server
- Mga environment variable at configuration
