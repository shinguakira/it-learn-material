# Task 5: Deploy

[← Bumalik sa Beginner Tasks](../index.md)

I-publish ang resume site mo sa internet para makita ng kahit sino.

## Mga Task

### Option A: GitHub Pages (inirerekomenda para sa mga baguhan)

- [ ] Pumunta sa repository mo sa GitHub
- [ ] I-click ang **Settings → Pages**
- [ ] Sa "Source", piliin ang **Deploy from a branch**
- [ ] Piliin ang **main** branch at **/ (root)** folder, tapos i-click ang **Save**
- [ ] Maghintay ng ilang minuto, tapos bisitahin ang `https://yourname.github.io/your-repo/`
- [ ] I-confirm na live na ang resume page mo sa internet
- [ ] I-share ang URL sa isang tao

### Option B: Vercel (alternatibo)

- [ ] Pumunta sa [vercel.com](https://vercel.com/) at mag-sign up gamit ang GitHub account mo
- [ ] I-click ang **Add New → Project**
- [ ] I-import ang GitHub repository mo
- [ ] I-click ang **Deploy**
- [ ] Bisitahin ang ibinigay na URL at i-confirm na live ang site mo

## Dapat Mong Maintindihan

- Ang deployment ay paggawa ng site mo na accessible sa internet
- Ang GitHub Pages ay nagho-host ng static na site (HTML/CSS/JS) nang libre
- Tuwing mag-`git push` ka, ang GitHub Pages ay pwedeng awtomatikong i-update ang site mo
- Ang URL ng site mo ay base sa GitHub username at repository name mo

## Congratulations!

Nakumpleto mo na ang lahat ng beginner tasks. Kaya mo na ngayon:
- Magsulat ng HTML para i-structure ang page
- Gumamit ng CSS para i-style ito
- Magdagdag ng interactivity gamit ang JavaScript
- I-track ang code mo gamit ang Git at GitHub
- I-deploy ang site mo para makita ng buong mundo
