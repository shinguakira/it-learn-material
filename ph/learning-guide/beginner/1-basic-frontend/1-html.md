# Task 1: HTML

[← Bumalik sa Beginner Tasks](../index.md)

> Sanggunian: [HTML](../../../frontend/html/) — Alamin ang tungkol sa HTML structure at tags

Gawin ang unang web page mo gamit ang HTML.

## Mga Task

- [ ] Gumawa ng bagong file na `index.html`
- [ ] Isulat ang basic HTML structure (`<!DOCTYPE html>`, `<html>`, `<head>`, `<body>`)
- [ ] Magdagdag ng `<title>` sa loob ng `<head>`
- [ ] Buksan ang file sa browser (i-drag & drop o gamitin ang Live Server)
- [ ] Gumawa ng personal na resume page na may:
  - [ ] `<h1>` para sa pangalan mo
  - [ ] `<h2>` para sa section title (Contact, Summary, Education, Skills)
  - [ ] `<p>` at `<strong>` para sa text content
  - [ ] `<ul>` at `<li>` para sa skill list
  - [ ] `<section>` para i-grupo ang bawat section
- [ ] Magdagdag ng `<link rel="stylesheet" href="style.css" />` sa loob ng `<head>` (gagawin natin ang CSS file sa susunod na task)

## Dapat Mong Maintindihan

- Ang `<head>` ay para sa metadata (hindi nakikita sa page)
- Ang `<body>` ay ang pangunahing nakikitang content
- Ang mga tag ay magkapares: `<tag>content</tag>` o self-closing: `<br>`, `<img />`
- Ang `<div>` at `<section>` ay nag-ggrupo ng content
