# Task 0: I-install ang VS Code

[← Bumalik sa Beginner Tasks](../index.md)

I-setup ang development environment mo bago magsulat ng code.

## Mga Task

- [ ] I-download at i-install ang [VS Code](https://code.visualstudio.com/)
- [ ] Buksan ang VS Code at tingnan ang interface (sidebar, terminal, explorer)
- [ ] I-install ang "Live Server" extension (para ma-preview ang HTML sa browser na may auto-reload)
- [ ] Gumawa ng folder para sa mga project mo (hal., `my-projects`)
- [ ] Buksan ang folder sa VS Code gamit ang **File → Open Folder**
- [ ] Buksan ang built-in terminal gamit ang **Ctrl + `** (o **View → Terminal**)
- [ ] Gumawa ng bagong file na `hello.html` at i-type ang `Hello World`
- [ ] I-right-click ang file at piliin ang **Open with Live Server** para makita ito sa browser

## Mga Tip

- Ang VS Code ay libre at gumagana sa Windows, Mac, at Linux
- Ang terminal sa loob ng VS Code ay pareho sa system terminal mo — pwede kang mag-run ng commands doon
- Pwede kang mag-install ng dagdag na extension mamaya kung kinakailangan (hal., Prettier para sa formatting)
