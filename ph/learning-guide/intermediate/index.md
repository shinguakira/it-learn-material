# Mga Intermediate Task

[← Bumalik sa Gabay sa Pag-aaral](../README.md)

Palakasin ang iyong pundasyon gamit ang mas advanced na mga challenge.

## Mga TypeScript Task

> Sanggunian: [TypeScript](../../frontend/typescript/) — Alamin ang tungkol sa typed JavaScript

- [ ] Mag-setup ng TypeScript project gamit ang `tsconfig.json`
- [ ] I-rewrite ang isang JavaScript file sa TypeScript na may type annotation
- [ ] Gumawa ng interface para sa data model (hal., User, Product)
- [ ] Gumamit ng generics sa utility function

## Mga React Task

> Sanggunian: [React](../../frontend/react/) — Alamin ang tungkol sa UI components

- [ ] Gumawa ng React app gamit ang `create-react-app` o Vite
- [ ] Gumawa ng reusable component (hal., Card, Button)
- [ ] Gumamit ng `useState` para pamahalaan ang component state
- [ ] Gumamit ng `useEffect` para mag-fetch ng data mula sa API
- [ ] Magdagdag ng routing gamit ang React Router

## Mga Backend Task

> Sanggunian: [Node.js](../../backend/nodejs/) | [Express](../../backend/express/) | [Mga Database](../../backend/databases/)

- [ ] Gumawa ng basic na Node.js HTTP server
- [ ] Mag-setup ng Express app na may mga route
- [ ] Gumawa ng REST API na may GET, POST, PUT, DELETE endpoints
- [ ] Kumonekta sa database (PostgreSQL o MongoDB)
- [ ] Magdagdag ng input validation at error handling
