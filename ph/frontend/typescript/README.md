# TypeScript (JavaScript)

[← Bumalik sa Frontend](../README.md)

Ang TypeScript ay nagdadagdag ng static type checking sa JavaScript, na nakakatulong sa pag-detect ng mga error bago mag-run. Ang mga konsepto ng JavaScript ay kasama rin dito — ang TypeScript ay isang superset ng JavaScript.

## Mga Matututunan

- Mga variable, data type, at operator
- Mga function at scope
- DOM manipulation
- Asynchronous programming (Promises, async/await)
- ES6+ features
- Basic types at type annotations
- Interfaces at type aliases
- Generics
- Enums at utility types
- Pag-configure ng TypeScript projects
