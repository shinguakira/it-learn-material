# TypeScript

[← Bumalik sa Frontend](../README.md)

Ang TypeScript ay nagdadagdag ng static type checking sa JavaScript, na tumutulong sa paghahanap ng mga error bago mag-runtime.

## Mga Matututunan

- Mga pangunahing uri at type annotation
- Mga interface at type alias
- Generics
- Mga enum at utility type
- Pag-configure ng mga TypeScript project
