# Unit Testing

[← Bumalik sa Testing](../README.md)

Ang unit testing ay nagbe-verify na ang mga indibidwal na piraso ng code ay gumagana nang tama sa isolation.

## Mga Matututunan

- Mga pangunahing kaalaman at prinsipyo ng testing
- Jest at Vitest
- Pagsulat ng mga test case at assertion
- Mocking at stubbing
- Test coverage
